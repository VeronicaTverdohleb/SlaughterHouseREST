Animal Registration System

A RESTful Web Service for Animal Registration

API Endpoints

Endpoint Method Description
/animals GET Retrieve all registered animals
/animals POST Add a new animal
/animals/{registrationNumber} GET Retrieve animal by registration number
/animals/by-date/{date} GET Retrieve animals by arrival date (yyyy-MM-dd format)
/animals/by-origin/{origin} GET Retrieve animals by origin (i.e., farm)
